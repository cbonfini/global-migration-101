# Global Migration 101

An interactive, WCAG 2.1 AA–accessible teaching course for university faculty —
14 modules on global migration with narrated lecture decks, knowledge checks,
faculty handouts, a 128-term glossary with flashcards, and progress tracking.

Part of **Refugees & Migrants in Our Common Home**, a Villanova University
initiative of the Mother Cabrini Institute on Immigration.

## Hosting

This is a fully static site — no build step. Serve the folder as-is
(GitHub Pages, Netlify, any web server) and point visitors at `index.html`.

## Structure

- `index.html` — course home with progress tracking
- `modules/` — 14 module pages with knowledge checks
- `presentations/` — narrated lecture decks (audio in `presentations/audio/`)
- `handouts/` — printable faculty handouts
- `glossary/` — searchable glossary + flashcard study mode
